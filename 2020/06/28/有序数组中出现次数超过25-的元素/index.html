LeetCode1287. 有序数组中出现次数超过25%的元素

给你一个非递减的有序整数数组，已知这个数组中恰好有一个整数，它的出现次数超过数组元素总数的25%。
请你找到并返回这个整数。

示例:

输入: arr = [1,2,2,6,6,6,6,7,10]
输出: 6

提示:

  • 1 <= arr.lengthk <= 10^4
  • 0 <= arr[i] <= 10^5

答案看这里哦👇

第一种方法，hash表，记录每个元素出现的次数，然后计算出现次数超过25%的元素

第二种方法优化: 一个for循环

第三种方法: 使用filter，但是速度稍慢
